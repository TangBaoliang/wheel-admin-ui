![https://github.com/yunlong826/wheel4j/tree/develop](https://img.shields.io/badge/后端项目-wheel4j-green.svg)

## 项目简介
这是一个基于 ruoyi-vue 的项目, 是一个管理监控 zookeeper 节点信息的系统。
此项目开发坚持一切从简的原则

## 技术栈
Vue2 + ElementUI + axios

## 项目启动
命令行运行
`npm run dev`
